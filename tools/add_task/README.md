### `add_task` — 添加定时任务
- **用途**：当你需要安排一个未来的任务（如提醒、定时操作）时，使用此工具。它会在 `tasks.json` 文件中记录任务，由后台程序在指定时间触发。
- **输入**：
  ```json
  {
    "content": "任务描述，例如 '提醒我去超市'",
    "time": "执行时间，格式建议 YYYY-MM-DD HH:MM 或 HH:MM（每天重复）"
  }
  ```
  - `content`：字符串，任务的具体内容。
  - `time`：字符串，任务执行的时间点。推荐使用标准格式（如 `2026-03-21 10:00` 表示一次性任务，`14:30` 表示每天重复）。
- **输出**：成功添加后返回任务 ID 和确认信息，如 `"任务已添加！ID: 1, 内容: 提醒我去超市, 时间: 14:30"`。若失败则返回错误信息。
- **注意事项**：
  - 任务会保存在项目根目录的 `tasks.json` 文件中，格式为 `{"ID": ["任务内容", "时间"]}`，ID 自动递增。
  - 请确保 `time` 参数符合后台调度程序能解析的格式，避免因格式错误导致任务无法执行。
